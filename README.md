# RemoveBG Pro

RemoveBG Pro is a web application that allows users to upload images and remove the background automatically using AI. It is powered by Flask for the web framework and `rembg` for the background removal process. The application is designed to be simple, user-friendly, and fast, providing an instant background removal service.

**Live Demo:** [RemoveBG Pro](https://removebg-pro.onrender.com/)

## Features

- Upload an image to remove its background automatically.
- View the original and processed images side by side.
- Download the processed image.
- Simple, intuitive, and fast UI for image processing.

## Technologies Used

- **Flask**: Web framework for building the application.
- **rembg**: AI-powered background removal library.
- **Pillow**: Image processing library to handle and manipulate images.
- **Jinja2**: Templating engine used by Flask for rendering HTML pages.
- **Gunicorn**: Production WSGI server to run the application.
- **Bootstrap**: For styling the front end with responsive design.
